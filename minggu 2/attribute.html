HTML attributes

Semua tag HTML bisa mempunyai attribute

Attribute menyediakan informasi tambahan

Attribute selalu ditentukan pada tag awal

Attribute biasanya berpasangan antara nama/nilai seperti name="value"

Attribute gambar

Attribute src untuk menentukan path di mana gambar disimpan

Attribute width dan height untuk menentukan lebar dan tinggi gambar

Attribute alt untuk menampilkan teks alternatif pengganti gambar yang tidak bisa dimuat

Attribute style

Attribute style untuk menambah style pada elemen, seperti warna, font, ukuran, dan lain-lain

Attribute title

Attribute title untuk menambah informasi tentang elemen. Nilai dari attribute title ditampilkan sebagai tooltips ketika mouse digerakan di atas elemen
